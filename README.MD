## ✨ Features (MVP)
- Electron overlay (always-on-top toggle)
- Modes: **Assist / Follow / Vibe**
- 🔊 **Text-to-Speech** replies + 🎙️ **Push-to-Talk** (experimental)
- Local FastAPI endpoint `POST /plan` (returns a suggestion)
